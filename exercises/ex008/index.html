Main Formats

Bold / Highlight

On this phrase we have a BOLD TERM using the B tag (Non-Semantic).

Now, on this phrase we have a BOLD TERM using the STRONG tag (Semantic).

Italic / Enphasis

Here we have a ITALIC TERM using the I tag (Non-Semantic).

And now, we have a TERM IN EMPHASIS using the EM tag (Semantic).

Marked Text

We can also create an MARKED TEXT using the MARK tag

Big and Small Texts

We are creating a BIG TEXT and a SMALL TEXT on this paragraph.

Deleted Text

We can mark a TEXT AS DELETED to indicate that it should be read, but not considered.

Inserted Text

We can mark a TEXT AS INSERTED to give it emphasis and to indicate that it was added after.

There's also the UNDERLINE with the U tag (Non-Semantic).

Superscript Text

To insert things like: X20+3

Subscript Text

To insert things like: H2O
